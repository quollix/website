The first public release of Quollix is now available.

Quollix is a self-hosting platform for running web applications with integrated user management, access policies, backups, automatic updates, HTTPS certificates, and app single sign-on.

For an overview, see the [home page]({{< relref "docs/_index.md" >}}).
