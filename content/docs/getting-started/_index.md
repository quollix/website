title: "Getting started"
weight: 20
---

Before you start, install Docker on the machine where you want to run Quollix. For general installation instructions, see the [Docker installation guide](https://docs.docker.com/get-docker/).

For production servers, we recommend Ubuntu and Docker Engine installation through `apt`. Follow Docker's [Ubuntu installation guide](https://docs.docker.com/engine/install/ubuntu/#install-using-the-repository).

Choose the local setup for testing, or follow the production setup when you are ready to run Quollix on a server.
