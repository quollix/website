The official Quollix Docker container includes both open-source and proprietary code.

Quollix has a Community Edition core and Premium Edition extensions. The Community Edition is open source and licensed under the [AGPLv3](https://opensource.org/license/agpl-v3). Using Quollix without registering a license key means using it as an open-source product.

The Premium Edition adds [premium features]({{< relref "../introduction/features.md" >}}) that are proprietary and licensed separately. If you enable premium features by [purchasing]({{< relref "../pricing/purchase-premium.md" >}}) and entering a license key, separate terms apply to those premium features. More information can be found in the [Legal]({{< relref "../legal/_index.md" >}}) section.
