Milestones for upcoming releases:

* **Complete feature set**: Implementing [all planned features]({{< relref "docs/introduction/features.md" >}}) for the current major version.
* **Managed SaaS offering**: Providing an optional hosted Quollix service for users who prefer not to self-host.
* **Official app expansion**: Adding more [official apps]({{< relref "docs/introduction/apps.md" >}}) to the App Store.
* **Third-party app publishing**: Publishing the App Store source code under an open source license and allowing third-party maintainers to register with the App Store to distribute apps.
