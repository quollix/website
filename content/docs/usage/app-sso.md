---
title: App Single Sign-On
---

When you install an app, Quollix automatically generates OIDC client credentials for browser-based sign-in using the authorization code flow. If the app supports this flow, users can sign in with their existing Quollix account. In the ideal setup, one central Quollix account can be used to sign in to every app.

Some apps are configured automatically to use these credentials and provide OIDC sign-in when they start. Other apps require manual configuration. For app-specific guidance, see the list of official apps [on the App Store usage page]({{< relref "docs/usage/app-store.md">}}).
