This glossary defines preferred wording so terms are used consistently across the Quollix ecosystem.

| Term | Preferred Use |
|------|---------------|
| License key | Customer-facing credential used to activate Premium in a Quollix instance. |
| Signed license token | Signed proof returned by the license server and cached by Quollix for Premium validation. |
| License server | Quollix service that validates license keys and issues signed license tokens. |
